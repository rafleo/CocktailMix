# CocktailMix
A "ThermoMix" for cocktails on Arduino base.

Display:
Nextion NX3224T024_011R
https://nextion.itead.cc

Scale sensor:
HX711AD Sensor 24 bit AD module

Sensor library:
https://github.com/bogde/HX711
